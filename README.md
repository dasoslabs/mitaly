<h3>프로젝트 실행</h3>

GitHub 저장소를 clone하고 해당 디렉토리에서 터미널을 열고 아래 명령어를 차례로 입력해주세요.

```
$ npm i
$ npm run dev
```

npm을 사용할 경우 그대로 진행하시고, yarn을 사용할 경우 npm 대신 yarn을 입력해주세요.

프로젝트는 [http://localhost:3000](http://localhost:3000)에서 실행됩니다.
